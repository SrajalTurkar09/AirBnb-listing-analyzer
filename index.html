Airbnb Listings Analyzer

Powerful data analysis tool for exploring and understanding Airbnb listing data.
Upload your listings.csv file and get instant insights!

Features

  • Automatic data cleaning and preprocessing
  • Median and mean price analysis by neighborhood and room type
  • Minimum and maximum price statistics
  • Room type distribution
  • Outlier detection using IQR method
  • Interactive visualizations: histograms, boxplots, bar charts, and outlier plots
